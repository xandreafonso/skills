# Site pessoal — Alexandre Afonso

Landing page premium, dark-first. Estrutura inspirada em sites de consultoria estratégica de boutique, recortada para perfil individual.

## Componentes
- `Nav.jsx` — barra superior fina, fixa, wordmark à esquerda, links no centro, CTA à direita
- `Hero.jsx` — tagline em display, lede, dois CTAs (primário/secundário), bloco com 3 KPIs
- `About.jsx` — duas colunas, retrato (placeholder) + bio + skills tags
- `Process.jsx` — 3 etapas numeradas em linha (Diagnóstico → Implementação → Operação)
- `Cases.jsx` — grid de cases recentes com badges e métricas
- `Manifesto.jsx` — bloco editorial com citação grande e listas curtas
- `CTA.jsx` — chamada para agendar diagnóstico
- `Footer.jsx` — mínimo, com wordmark, contato e copyright

## Como abrir
`index.html` na raiz da pasta carrega tudo via Babel standalone. As fontes vêm do Google Fonts (Geist/Geist Mono).
